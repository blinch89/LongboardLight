Longboardlight

An attempt to create LED-powered light effects under a longboard deck.

Download this project as a .zip file Download this project as a tar.gz file

Hardware

  • LEDs: a strip of WS2812 or WS2811, 30LEDs/meter
  • microcontroller: XMC2GO (ARM Cortex M0)
  • power supply: 2,2Ah USB power bank, max. current 1A
    (not enough, when all 21 LEDs are white at full brightness, Voltage drops to 3.4V but everything is working fine)

Firmware

  • WS2812 driver library from CPLDCPU, https://github.com/cpldcpu/light_ws2812
  • switching between different modes or color patterns via button
